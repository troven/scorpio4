Scorpio4 - Maven

The scorpio4maven plugin provides a suite of helpful tools for dealing with RDF-based technologies.

  • API

    Launches the scorpio4API sandbox. The API sandbox is a developer-runtime that is backed by a Sesame Repository. The API runtime offers a variety of scripting, templating and data-flowing features.

  • Assets

    Scans the [scorpio.source] directory looking for supported text files, called Assets. Scorpio4 Assets includes SPARQL & SQL queries, Groovy & Javascript, XHTML & HTML, CSV and XML file formats

  • Check

    Checks the local Repository for errors using appropriately declared SPARQL Rules that conform to the FactCheck conventions.

  • Execute

    Executes the declared FactTool commands from the local Repository.

    • Scripting
    • Inferencing
    • Templating

  • Publish

    Generate new Assets by rendering a template for each result of the associated query. The resulting Assets are stored in the local Repository.

  • Infer

    Perform backward chained inferencing using appropriately declared SPARQL Rules. The resulting Assets are stored in the local Repository.

  • x

    RDF

    Loads the RDF files stored in the [scorpio.source] directory into the local Repository. The local Repository is emptied before the operation begins.

  • Dump

    Outputs the working repository into the [scorpio.temp] directory.

  • Upload

    Uploads the contents of the working repository into a remote Sesame Repository or SPARQL endpoint.
