Execution Pipeline

An Execution Pipeline is a list of scripts that execute sequentially

Scorpio offers three powerful command patterns that can be access by the Maven Execute command.

  • Scripting

    Scorpio provides a rich scripting environment that ships with Groovy and Javascript. Other programming languages such as Python, Ruby, LUA, AppleScript and R are available as plugins.

  • Templating

    Scorpio can generate text-based Assets from a set of templates. Each Templating chain comprises a query and a template that is rendered for each query result

  • Inferring

    Scorpio supports SPARQL Rules that perform backward-chaining inferencing.
