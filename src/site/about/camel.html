FLO (Fact Liberation Ontology)

FLO is an Active Vocabulary that describes Enterprise Integration Patterns, and how to instantiate them.

Modern businesses need to integrate with external services, using web APIs or legacy technologies

Apache Camel is a lightweight integration framework which implements all of Enterprise Integration Best Practices.

Thus, you can easily integrate different applications using appropriate patterns. Scorpio4 provides a semantic modelling language for Apache Camel, called FLO.

Apache Camel exchanges, routes and transforms data using various protocols. It includes components for various backend systems along with powerful decision making, routing and filtering.

You can use Java, Spring XML, Scala or Groovy. Almost every technology you can imagine is available, for example HTTP, FTP, JMS, EJB, JPA, RMI, JMS, JMX, LDAP, Netty, and many, many more.
