Active Beans

Active Beans is an Active Vocabulary to manage the lifecycle of Java objects

The ubiquitous Spring IoC framework can be used to manage the lifecycle and configuration of Scorpio4 application components

Spring can be used to contextually configure data access objects, user interfaces as well as to integrate other components and libraries into the Scorpio4 runtime environment.

At runtime, the Java object graph is constructed by interpreting semantic data models. These models can be inferred and derived from existing domain knowledge.

Bean Ontology

Scorpio4 implements the Bean Ontology to configure and register bean definitions. At runtime, these definitions are resolved to construct the appropriate runtime Java environment as required.

The current integration is substantially functional, complex cases and more esoteric features are not yet supported.
