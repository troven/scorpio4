Scripting Support

Scorpio4 includes a selection of scripting languages by default.

All scripting languages are supported by all Active Vocabularies.

Default Languages

  • Groovy
  • JavaScript (ECMAScript)
  • Python (Jython)
  • Ruby (JRuby)
  • R (Renjin)
  • Java
  • AppleScript (OS X)

Optional Languages

  • AWK
  • BeanShell
  • ejs
  • FreeMarker
  • Jaskell
  • Jelly
  • JEP
  • Jexl
  • jst
  • JudoScript
  • JUEL
  • OGNL
  • Pnuts
  • Scheme
  • Sleep
  • Tcl
  • Velocity
  • XPath
  • XSLT
